What is node.js:
Node is server side Javascript. Build on Google's V8. It's non-blocking.

What is a callback?
A function that is used as a function parameter.

What is a variable?
"Pointer" to a memory location.

Why do we modularize?
Good for...
	* Organization
	* Collaboration / delegating work

What is an object?
 - Contains one or more key/value pairs.
 - Can contains "anything" from other objects to strings to functions, etc.
 - Collection of properties.

What is a framework?
 - Set of tools
 - Collection of functions built to solve a particular set of problems.
 - Sometimes split into front and back-end framework.
